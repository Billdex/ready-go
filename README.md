# ready-go
go-web后端项目骨架，组合好常用工具，用来快速启动项目

# 项目结构

```
├── app                 // 业务相关的主要代码
│ ├── dao               // 数据库的访问操作
│ ├── http              // http接口的业务处理
│ │ ├── api             // 接口层，解析参数与业务逻辑调度
│ │ ├── middleware      // 中间件
│ ├── model             // 数据库实体结构定义
│ └── service           // 实现详细业务逻辑
├── bootstrap           // 执行项目启动时的环境初始化
├── config              // 进行配置文件解析与存放示例配置
├── cmd                 // 程序入口，存放 main.go 与一些启动脚本
├── docs                // 项目相关文档
├── pkg                 // 封装功能的工具包，不包含具体业务实现
├── router              // 路由统一注册管理
└── go.mod
```
